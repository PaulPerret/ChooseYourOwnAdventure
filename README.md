# Choose Your Own Adventure

A simple Java project for a choose-your-own-adventure game.

## Prerequisites

- Java 11 or higher
- Maven

## Running the Project

To compile and run:

```bash
mvn compile exec:java
```

## Project Structure

- `src/main/java/` - Main source code
- `src/test/java/` - Test source code
- `pom.xml` - Maven configuration
